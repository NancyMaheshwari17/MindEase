# MindEase – Mental Wellness Website 🌿

## Project Overview

MindEase is a simple **mental wellness website** designed to help people relax, stay positive, and practice mindfulness.

In today’s fast-paced world, stress and overthinking are common problems. MindEase provides a calm digital space where users can explore motivational quotes, wellness tips, and beginner-friendly yoga practices to improve their mental health.

This project demonstrates front-end web development using **HTML and CSS** while focusing on a meaningful real-world theme — mental well-being.

## Features

* 🌿 Clean and calming user interface
* 💬 Motivational quotes to inspire positivity
* ✉️ Wellness tips for mental health
* 🧘 Yoga section for relaxation and mindfulness
* 📞 Contact section
* 📱 Responsive layout for better viewing

## Website Sections

The website contains the following pages:

* **Home** – Introduction to MindEase and its purpose
* **Quotes** – Inspiring quotes to motivate users
* **Tips** – Practical tips for improving mental wellness
* **Yoga** – Simple yoga practices for relaxation
* **Contact** – A section for connecting with the website

## Technologies Used

* HTML5
* CSS3
* Font Awesome Icons
